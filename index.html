前端工具學習

GitHub

開發者能公開專案，任何人都能存取、使用、甚至對專案的內容做出修改，而專案作者能對這些改動用 Git (等等會介紹的版本控制工具)來進行控管，進而達成一種「共同創作」的理念。

Node.js

Node.js 是能夠在伺服器端運行 JavaScript 的開放原始碼、跨平台 JavaScript 執行環境。

Bootstrap

作為CSS的框架，有更完善的檔案、提高開發效率以及更便於維護。

Webpack

他是在node.js的環境下執行的工具，也可以透過npm下載其他webpack的套件來使用。他是一個幫助程式碼解析成瀏覽器可以閱讀的格式、統一開發環境、可以把引入多個source時打包成一個檔案
